## 🚀 Quantum Project Log

<details>
  <summary><b>Streamy (Decentralized Media Hub)</b></summary>
  <br/>
  - **Description:** A robust backend for a YouTube-like platform, supporting advanced video processing, dynamic comments, user-centric interactions (likes/dislikes), and a scalable subscription framework.
  - **Technology:** Engineered with `Node.js`, `Express.js`, `MongoDB` for data persistence, and `Mongoose` for object data modeling.
  - **Features:** Implemented secure, token-based authentication using `JWT` and `Bcrypt` for enhanced user privacy, integrated with a dynamic subscription system enabling diverse content delivery models.
</details>

<details>
  <summary><b>Synaptic Health Gateway (AI-Powered Wellness System)</b></summary>
  <br/>
  - **Description:** A full-stack AI-driven application designed to analyze and classify mental health statements into five distinct conditions with a predictive accuracy of 91%.
  - **Technology:** Frontend interfaces crafted with `Next.js (TypeScript)` and styled using `Tailwind CSS`, featuring interactive user experiences powered by `Framer Motion`. The intelligent backend processing unit utilizes a `BERT` model for sophisticated natural language understanding.
  - **Features:** Incorporates a dynamic, real-time dashboard for intuitive visualization of assessment outcomes, promoting higher user engagement and insightful data interpretation.
</details>
